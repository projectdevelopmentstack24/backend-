 api option - 
Get Balance - user balance get api 
http://api.localhost.in/balance/apikey={api-key}
response - {"balance":100} agr 0 balance hai to
{"balance":0}
other response - 
{"error":"invalid key"} - wrong api key
{"error":"under maintenance"} - if website under maintenance 

Get Number - get number ka api hoga like
http://api.localhost.in/number/apikey={api-key}&server={servernumber}&servicename={servicename}
responce - {"status":"ok","number":"9999999999","id":"123456789"}
other responce - 
{"error":"invalid key"} - wrong api key
{"error":"no stock"} - when number not available 
{"error":"low balance"} - when balance low
{"error":"under maintenance"} - if website under maintenance 

Get OTP - otp get api 
http://api.localhost.in/otp/apikey={api-key}&server={servernumber}&id={id}
responce - 
{"status":"ok","otp":"waiting for otp"} - waiting for otp
{"status":"ok","otp":"1235 is your otp"}- when otp come 
{"status":"ok","otp":"number cancelled"} - if number already cancelled 
{"error":"under maintenance"} - if website under maintenance 

Cancel Number - number cancel api 
http://api.localhost.in/cancel/apikey={api-key}&server={servernumber}&id={id}
responce - 
{"status":"success"}- when number is cancel 
{"error":"wait 2 mints before cancel"} - if user try to cancel number before 2 mint
{"error":"under maintenance"} - if website under maintenance 
{"error":"otp already come"} - if otp came and user run number cancel api 

Get Service List and Server List 
http://api.localhost.in/list-price/apikey={api-key}
{"error":"under maintenance"} - if website under maintenance
